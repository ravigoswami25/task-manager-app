About this app

Task Manager is a small Angular app for creating, listing, and removing tasks. It is a clear example of standalone components, routing, and client-side persistence without a backend.

What you can do

  • Add tasks on the home page; the list updates right away and data is saved in your browser.
  • Delete tasks when you are done with them.
  • Move between Home and this About page using the header links.

How it works

Tasks are stored in localStorage under a fixed key. Each browser profile has its own storage, so nothing is shared between users or devices. Refreshing the page keeps your tasks; clearing site data removes them.

A shared TaskService reads and writes that storage and notifies the task list when data changes, so the UI stays in sync after add or delete without a full reload.

Tech stack

Angular · TypeScript · localStorage
